# RaceTrace Articles
 
Data-driven F1 practice session analysis articles, powered by the **aggregated vs actual lap time** methodology.
 
These articles are published on [RaceTrace](https://racetrace.tech) and break down Formula 1 free practice sessions to uncover real pace trends beyond the raw timing sheets.
 
## What's Inside
 
Each article analyzes practice session data - comparing aggregated sector times against actual posted laps - to surface insights that traditional timing summaries miss. This methodology highlights where drivers are genuinely quick versus where a single hot sector flatters their position.
 
Articles are written in a structured format, ready for integration with the RaceTrace platform.
 
## Methodology
 
**Aggregated vs Actual Lap Time** - instead of relying solely on a driver's best lap, this approach takes each driver's best individual sector times and combines them into a theoretical best. Comparing this aggregated time to the actual best lap reveals how complete a driver's session was, and where hidden pace might be lurking.
